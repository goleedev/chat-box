<!-- GETTING STARTED -->
## Getting Started

1. Clone the repo
```sh
git clone https://github.com/goleedev/chat-box.git
```
2. Change directory to server
```sh
cd server
```
3. Install NPM packages
```sh
npm install
```
4. Run server
```sh
node index.js
```
5. Change directory to client
```sh
cd ..
cd client
```
6. Install NPM packages
```sh
npm install
```
7. Start app
```sh
npm run start
```
